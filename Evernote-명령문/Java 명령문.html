# String

-  intern()
   일반 스트링 인스턴스의 값을 상수 스트링 풀에 생성
   ex> String s1 = new String(“aaa”);
          String s2 = s1.intern();

-  substring(int beginIndex) , subString(int beginIndex, int ednIndex)
   문자열에 대해 인덱스 번호를 주어 해당 문자열 출력 (리턴값 string)

- indexOf(ch), indexOf(String)
  해당 문자 또는 문자열의 인덱스 번호를 리턴 (리턴값 int)

- split()
  문자열을 파라미터 값 대상으로 잘라서 각각의 변수를 생성하고 해당 주소를 담는 배열을 리턴 해준다. (리턴값 string[])
  ex> String[] arr = “ABC DEF GHI”.split(“ “); -> 공백을 기준으로 잘라서 배열 길이가 3인 배열을 리턴한다.

- trim()
  문자열중 공백을 날려준다. 입력 값중에 공백이 있어도 공백을 제외하고 문자열 만 남긴다.

 - String -> char로 변환   charAt
    ex> String s = "A";  or  String[] s = {"A"}
           char c = s.charAt(0);  or  char c = s[0].charAt(0);

 -String -> char[] 변환   toCharArray()
 - String[] -> char[] 변환
    ex> String str = "ABCDEFG"; or Stirng[] str = {"A"};
           char[] ch1 = str.toCharArray(); or str[0].toCharArray();

    ch1의 배열이 생성 된다. 
    ch1 -> { 'A', 'B', 'C', 'D', 'E', 'F', 'G' }

 - 문자열을 대문자 or 소문자로 변환

소문자로 만들기  -> toLowerCase()     -> str.toLowerCase()
대문자로 만들기  -> toUpperCase()     -> str.toUpperCase()

 - 문자열 비교 방법
    == -> 사용 불가
   .equals("ABC")
   DataType이 Char인 경우는 == 사용 


  # Math

 - 절대값 구하기.  => Math.abs(값)
 - 랜덤변수 구하기 => Math.random()
